Gerar ícones PNG do app

Esta página gera dois arquivos PNG (192×192 e 512×512) com a identidade visual do app. Você só precisa abrir uma vez, baixar e subir pro GitHub.


icon-192.png

icon-512.png

Passo a passo

  1. Click no botão acima — vai baixar icon-192.png e icon-512.png automaticamente.
  2. No GitHub, abre a pasta icons/ do repositório.
  3. Click no botão Add file → Upload files.
  4. Arrasta os 2 PNGs pra dentro.
  5. Commit com mensagem "Adicionar ícones PNG".
  6. Pronto. O manifest já está atualizado pra usá-los automaticamente.
